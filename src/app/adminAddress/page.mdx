# Admin address 

Os address são uma parte essencial do sistema administrativo — eles contêm informações necessárias para localizar usuários, clientes ou recursos no sistema. Nesta página, exploraremos os diferentes endpoints relacionados a admin address  que você pode usar para gerenciá-los programaticamente. Veremos como consultar, listar e detalhar address.

## O modelo de admin address 

O modelo de admin address  contém todas as informações detalhadas sobre os address armazenados no sistema, como CEP, rua, bairro, estado e número da casa. Essas informações são usadas para gestão e localização.

### Propriedades

| Nome                | Tipo       | Descrição                                             |
|---------------------|------------|-----------------------------------------------------|
| **zipCode**         | string     | CEP do endereço.                                     |
| **street**          | string     | Nome da rua ou avenida.                              |
| **neighbourhood**   | string     | Bairro do endereço.                                  |
| **state**           | string     | Estado onde o endereço está localizado.             |
| **city**            | string     | Cidade onde o endereço está localizado.             |
| **houseNumber**     | string     | Número da casa ou estabelecimento.                  |
| **complement**      | string     | Complemento do endereço, se aplicável.              |
