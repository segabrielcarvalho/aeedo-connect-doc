# Address

Os address são uma parte essencial do sistema administrativo — eles contêm informações detalhadas sobre os locais associados a usuários ou entidades no sistema. Nesta página, exploraremos os diferentes endpoints relacionados aos address que você pode usar para gerenciá-los programaticamente. Veremos como consultar, criar, atualizar e validar address.

## O modelo de address

O modelo de address contém todas as informações necessárias para identificar um local, incluindo CEP, rua, bairro, estado, cidade, número da casa e complementos.

### Propriedades

| Nome             | Tipo    | Descrição                                      |
|-------------------|---------|------------------------------------------------|
| **id**           | string  | Identificador único do endereço.               |
| **zipCode**      | string  | Código postal (CEP) do endereço.               |
| **street**       | string  | Nome da rua ou avenida.                        |
| **neighbourhood**| string  | Bairro onde o endereço está localizado.        |
| **state**        | string  | Estado onde o endereço está localizado.        |
| **city**         | string  | Cidade onde o endereço está localizado.        |
| **houseNumber**  | string  | Número da casa ou apartamento.                 |
| **complement**   | string  | Informação adicional sobre o endereço (opcional). |

## POST Criar Endereço {{ tag: 'POST', label: '/addresses' }}

<Row>
  <Col>

    Este endpoint permite criar um novo endereço associado a um usuário.

    ### Corpo da Requisição (Request Body)

    ```json
    {
        "zip_code": "97541-370",
        "street": "Praça Marechal Costa e Silva",
        "neighbourhood": "Centro",
        "state": "RS",
        "city": "Alegrete",
        "house_number": "666",
        "complement": "",
        "user_id": "9da86d02-3686-42e3-8cff-83022baeed51"
    }
    ```

    ### Resposta Bem-sucedida (Successful Response)

    ```json
    {
        "data": {
            "zipCode": "97541370",
            "street": "Praça Marechal Costa e Silva",
            "neighbourhood": "Centro",
            "state": "RS",
            "city": "Alegrete",
            "houseNumber": "666",
            "complement": null
        }
    }
    ```

    ### Resposta de Validação (Validation Response)

    ```json
    {
        "message": "Este campo é obrigatório (and 5 more errors)",
        "errors": {
            "zip_code": ["Este campo é obrigatório"],
            "street": ["Este campo é obrigatório"],
            "neighbourhood": ["Este campo é obrigatório"],
            "state": ["Este campo é obrigatório"],
            "city": ["Este campo é obrigatório"],
            "user_id": ["Valor inválido ou inexistente"]
        }
    }
    ```

  </Col>
</Row>

## PATCH Atualizar Endereço {{ tag: 'PATCH', label: '/addresses/:id' }}

<Row>
  <Col>

    Este endpoint permite atualizar informações específicas de um endereço existente.

    ### Corpo da Requisição (Request Body)

    ```json
    {
        "house_number": "154"
    }
    ```

    ### Resposta Bem-sucedida (Successful Response)

    ```json
    {
        "data": {
            "zipCode": "97541370",
            "street": "Praça Marechal Costa e Silva",
            "neighbourhood": "Centro",
            "state": "RS",
            "city": "Alegrete",
            "houseNumber": "154",
            "complement": null
        }
    }
    ```

    ### Resposta de Validação (Validation Response)

    ```json
    {
        "success": false,
        "message": "Endereço não encontrado"
    }
    ```

  </Col>
</Row>

## GET Consultar Endereço por ID {{ tag: 'GET', label: '/addresses/:id' }}

<Row>
  <Col>

    Este endpoint permite consultar os detalhes de um endereço específico.

    ### Corpo da Requisição (Request Body)

    Nenhum corpo necessário.

    ### Resposta Bem-sucedida (Successful Response)

    ```json
    {
        "data": {
            "zipCode": "41535-204",
            "street": "Rua Luiz, 19. 41º Andar",
            "neighbourhood": "et",
            "state": "PB",
            "city": "Daniele do Norte",
            "houseNumber": "427",
            "complement": null
        }
    }
    ```

    ### Resposta de Validação (Validation Response)

    ```json
    {
        "success": false,
        "message": "Endereço não encontrado"
    }
    ```

  </Col>
</Row>
